Projects

Axion
github.com/georgelorentzos/axion →
Full-Stack  ·  Real-Time  ·  Production-Ready

Axion is a full-stack real-time communication platform built entirely from scratch. It supports direct messaging with reply threads, community servers with text channels and categories, a granular role and permissions system, live online presence, friend management, moderation tools, audit logs, and WebSocket-driven real-time updates across every feature. Every component — database schema, backend API, WebSocket layer, and frontend — was independently designed and built.

Reduced average API response time from 45ms to 7ms — a 6x improvement — by fully rewriting the backend from Python FastAPI to Go. The migration also involved refactoring the entire codebase for cleaner architecture, better separation of concerns, and significantly higher WebSocket concurrency. Fully containerized with Docker and built to scale horizontally.
